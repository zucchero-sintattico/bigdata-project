# bigdata-project

This repository contains the project for the Big Data course.

## Dataset
Download dataset:

```bash
#!/bin/bash
curl -L -o ~/Downloads/spotify-millions-playlist.zip\ https://www.kaggle.com/api/v1/datasets/download/adityak80/spotify-millions-playlist
```
